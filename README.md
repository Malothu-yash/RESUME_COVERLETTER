# Resume & Cover Letter Builder

A lightweight, browser-based tool for creating professional resumes and cover letters — no sign-up, no install, no data leaves your device.

## Features

- **Resume Builder** – fill in personal info, work experience, education, and skills with a live side-by-side preview.
- **Cover Letter Builder** – fill in recipient details and letter content with a live preview.
- **Print / Save as PDF** – use the browser's built-in print dialog to export a clean PDF.
- Zero dependencies – pure HTML, CSS, and vanilla JavaScript.

## Getting Started

1. Clone the repository or download a ZIP.
2. Open `index.html` in any modern web browser.
3. Fill in your details and click **Print / Save as PDF**.

## Project Structure

```
index.html      – Main application
css/style.css   – Stylesheet
js/app.js       – Application logic (live preview, print)
```
